# Projeto Fullstack - Node.js + Vue.js

Este projeto é dividido em duas partes: **Backend (Node.js + Express)** e **Frontend (Vue.js)**.  
O objetivo é fornecer uma base simples para aplicações fullstack com autenticação, banco de dados SQLite e interface em Vue.js.

---

## ⚙ Backend

### Dependências principais
- express
- sequelize
- sqlite3
- bcrypt
- jsonwebtoken
- cors

### Instalação
cd backend
npm init -y
npm install express sequelize sqlite3 bcrypt jsonwebtoken cors
npm install -g @vue/cli
vue create frontend
cd frontend
npm install vue vue-router vuex axios
